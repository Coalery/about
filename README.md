## About

자기 소개 페이지입니다. 이력서를 겸합니다.

Static Site를 생성하여 Github Pages로 서빙됩니다.

### Prerequisites

- Node.js v20.11.1

### Install

```shell
git clone https://github.com/Coalery/about.git
cd about
npm install
```

### Run in dev

```shell
npm run dev
```

### Build

```shell
npm run build # output: /out
```
